# Tiles

概述

Tiles 是Mac系统上的一款极简主义窗口管理器。

功能简介

· 通过将窗口拖动到屏幕的两侧或角落，将工作空间中的窗口抓取并组织起来，它们会自动调整大小，并固定在原地；
· 支持自定义的键盘快捷键，以提高工作效率。
