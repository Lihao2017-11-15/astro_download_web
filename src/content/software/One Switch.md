# One Switch

概述

One Switch 是 Mac 端的多合一开关合集工具，常驻顶部菜单栏，可以快速切换不同的设置，打开多种功能。一键切换，拒绝繁琐，One Switch 被海外媒体评选为 2019 最佳 Mac 软件。

功能简介

· 支持快速切换夜间模式，不用在设置面板中找来找去，直接点击 One Switch 上的「黑暗模式」，还能够根据需求，自定义切换时间。
· 一键隐藏桌面，不论是 PPT 工作汇报或是远程演示，随时随地保护个人隐私，并且在截图操作时，也能拥有更干净的桌面。
· 一键保持屏幕常亮
，不用安装额外的应用，关键时刻防止屏幕熄灭，还可自定义常亮时间。
·  支持快速连接蓝牙、屏幕清洁模式...目前 One Switch 支持 15 + 开关操作，除此之外，新的开关还将在迭代中不断增加。

## 推荐语

一款软件能解决多种需求，不用安装过多工具，还菜单栏干净整洁的状态。还可选择自己喜欢的菜单栏图标，更具个性化。
