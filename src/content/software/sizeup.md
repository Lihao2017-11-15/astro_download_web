# SizeUp

## 软件简介

## 详细介绍

概述

在 Mac OS X 上排列窗口既乏味又不精确。即使在拖动、拉动和重新拖动窗口之后，您最终也无法有效地利用可用的屏幕空间。SizeUp 将帮助您快速调整窗口的大小和位置，以充分利用您的屏幕。

功能简介

· SizeUp 带有许多预定义的操作，用于调整和/或移动几乎所有应用程序的最前面的窗口。Safari、Finder、Pages，没关系，SizeUp 会发挥它的魔力来帮助您将窗口固定到位；
· SizeUp 是高度可配置的，在屏幕边缘预留空间，这对于访问桌面图标非常有用；
· 在窗口之间添加一些距离以避免阴影重叠。
