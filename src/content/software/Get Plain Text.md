# Get Plain Text

概述

Get Plain Text Mac版是Mac上清除剪切板文本格式的工具，清除链接、颜色、字号等，粘贴为纯文本。

功能简介

· 将剪切板上的文字粘贴为纯文本
· 支持将（""）简化为（""），程序员们的福利！

## 推荐语

Mac上相当烦恼的一件事儿就是复制的文本带有格式，粘贴后还需要手动调整。Get Plain Text简单易用，可自动清除也可手动清除。
