# Get Plain Text

## 软件简介

“Mac上相当烦恼的一件事就是复制的文本带有格式，粘贴后还需要手动调整。它简单易用，支持自动清除文本格式，也可手动清除。”

## 详细介绍

概述

Get Plain Text Mac版是Mac上清除剪切板文本格式的工具，清除链接、颜色、字号等，粘贴为纯文本。

功能简介

· 将剪切板上的文字粘贴为纯文本
· 支持将（""）简化为（""），程序员们的福利！
