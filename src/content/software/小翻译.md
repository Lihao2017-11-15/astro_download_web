# 小翻译

概述

是一款简便的英语翻译软件，会实时监控您的剪贴板，复制即可自动翻译，并且实时展示翻译结果，非常方便，特别沉浸阅读外文资料的时候。

功能简介

· 支持中英文互译
· command + C 复制即可触发翻译，可以自定义翻译结果自动消失的时长，可以做到不打断当前阅读
· 无须翻译的时候可以选择退出，防止日常复制触发翻译

## 推荐语

区别与市面上的翻译词典软件，小翻译出发点是快速阅读外文资料，查看论文资料很方便，占用资料小反应快
