# 人人译视界

概述

人人译视界是一款智能协同翻译工具。由原人人影视核心创始团队研发，集翻译协作管理、AI辅助、字幕编辑、视频后期等功能于一身，一个软件解决所有视频翻译问题。

功能简介

· 字幕编辑、修改：支持逐句编辑和修改视频的字幕
· 时间轴调整：支持按照字幕拆分、合并等功能
· 字幕特效：支持更改字幕的字体、颜色、大小、甚至给字幕加边框、更改字幕位置等，都可以在后期工具的“样式管理器”中直接修改
辅助工具：人人译视界提供了一系列翻译协作辅助工具，如“术语库”、“在线纠错”、“查找”、“替换”、“名称列表”等。
