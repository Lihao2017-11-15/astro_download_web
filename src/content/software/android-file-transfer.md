# Android File Transfer

## 软件简介

Android File Transfer 是一款Mac上管理Androi手机的文件的软件，只需在连接手机前运行 Android File Transfer ，再连接手机，那么 Mac 就会识别出 Android 设备，并可以对里面的文件进行复制、粘贴、删除等管理。
