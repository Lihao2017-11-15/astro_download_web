# Pixeur

概述

Pixeur 是一个专为设计师和开发者设计的屏幕拾色工具，可以高效提升工作效率。

功能简介

· 丰富的颜色代码格式：覆盖主流平台及开发语言（iOS， Android，Swift，OC, CSS, Java 等），快捷键快速复制代码
· 拾色放大镜，支持聚焦模式，配合快捷键灵活取色
· 色块目录： 无限量的目录方便管理颜色
· 历史色块自动记录最近的色块
· 配色支持 RGB，HSB，CMYB，色盘，BOX
· 支持 iCloud 数据同步
