# Parallels Client

概述

Parallels Client 是一款免费的远程连接桌面工具，可以在Mac上远程连接服务器，实现远程操作服务器，Parallels Client还支持将本地文件直接复制到远程Windows服务器中。

功能简介

· Parallels Client for Mac 允许添加多个连接，且每个连接设置不同的属性
· Parallels Client支持将本地文件直接复制到远程Windows服务器中，共用剪切板
· 支持设备重定向，将声音从任何桌面重定向到Mac

## 推荐语

这款软件能解决你远控服务器的需求。无论是RDP，还是Parallels Remote Application Server，都能轻松连接，连接的流畅度甚至比微软自己出的都好！
