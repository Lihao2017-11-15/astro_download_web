概述

NXPowerLite Desktop 是一款文件体积压缩工具，支持对 PDF、Office 文档、图片等进行瘦身，拖拽文件即可处理，自定义压缩参数方便快捷。

功能简介

· 支持多种文件格式：可快速减小 PDF / Word / PowerPoint / Excel / JPEG / PNG / TIFF 文件体积；
· 批量处理更省心：单次可压缩近万个文件，满足快速压缩大量文件体积的需求；
· 自定义配置文件：根据不同需求创建压缩配置文件，可在软件首页快速调用提高效率；
· 右键菜单集成功能：可在资源管理器中直接点击右键菜单，快速进行文件瘦身压缩。
