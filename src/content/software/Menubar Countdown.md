# Menubar Countdown

概述

Menubar Countdown 是一款菜单栏倒计时应用，显示在macOS菜单栏上。这可以帮助你更好的管理任务的完成时间，更加的有效率。

功能简介

· 要设置计时器，单击Menubar Countdown的菜单栏图标并选择Start…菜单项。将出现一个对话框，允许您指定以小时、分钟和秒为单位的倒计时时间；
· 允许您指定以下哪一种形式的通知时，您想要定时器到00:00:00。只需一个点击菜单栏中的图标。倒计时倒数会有播放系统提示音。显示一个警告提示窗口给你。在通知中心显示通知。您可以指定要朗读的文本。
