# 网易邮箱大师

概述

网易邮箱大师是一款跨平台的邮箱管理应用，支持管理网易、QQ、Gmail、Outlook等个人邮箱以及超过一百万种企业邮箱，支持跨设备、跨平台云同步，支持日程管理提高工作效率，支持对机票、快递、银行等服务邮件以卡片的形式智能展示。

功能简介

· 友好的用户界面，同一主题的邮件智能合并显示
· 提供Windows、MacOS、手机等多个平台的客户端，通过云端实现跨平台无缝邮箱管理
· 支持网易、QQ、Gmail、Outlook等个人邮箱以及超过一百万种企业邮箱
· 对机票、快递、银行等服务邮件以卡片的形式智能展示


## 推荐语

全面支持网易、QQ、Gmail、Outlook 等主流个人邮箱，更有超过 1，000，000 种企业邮箱使用大师账号，多端同步信息。不同设备，一样的邮箱体验。多平台协同处理日程，你的工作安排，现在更井井有条机票、快递、银行账单邮件卡片化呈现，还可以一键提醒还款、通知快递动态。
