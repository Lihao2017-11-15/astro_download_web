# 专注笔记

概述

是一款精致的学习笔记软件，可折叠段落、多种高亮颜色、支持 LaTeX 数学公式、外框、拟真的笔记本及分页、分组管理...一切为了让您更简单地做出高可读性的学习笔记

功能简介

· 针对不同的标题段落，点击“H1/H2/H3”段落符号即可折叠当前段落。方便地进行大纲式的阅读及管理。
· 支持 Markdown 语法，大部分的功能可以通过快捷键触发。
· 支持 iCloud 云同步，在多个不同的终端同步文件。与此同时，还可以设置部分文件仅保存在设备中
· 多款编辑主题，多种字体可供使用

## 推荐语

Mac 及 iOS 上精致的学习笔记软件。曾多次在亚洲地区获得 apple 首页专题推荐。支持 LaTeX 、表格、折叠内容、更改文档主题、多种高亮颜色等功能。功能强大、高颜值、交互流畅优雅。
