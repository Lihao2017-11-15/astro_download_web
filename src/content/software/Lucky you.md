# Lucky you

概述

是一款跨平台抽奖软件，免费，开源，体积小巧，主要用于抽奖/课堂点名等需要从多个图片中随即选出一个的应用场景，使用简单。

功能简介

· 可以支持选中文件夹里面所有的图片和展示图片名称
· 声音控制：可以在 静音 / 非静音 状态之间切换，如果是静音状态，则点击 开始 和 停止 按钮时不会播放声音
· 语言切换：当前软件支持两种语言，中文 / 英文。
