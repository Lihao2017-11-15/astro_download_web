# 研报客

概述

一站式提供全球各行各业的研究报告阅读、下载、转换、翻译、推送。收录各行各业行业分析研究报告、公司调研调查报告、行业前景趋势预测报告、券商研报等。

功能简介

· 软件提供全球智库机构各行各业的行业研究报告文件，可以直接阅读并下载。
· 支持关键词搜索功能，可以通过关键词找到报告，同时具有全文搜索功能。
· 软件提供的文档都是PDF，保证可以正常打开。同时提供pdf转doc与ppt功能。
· 支持多标签同时查看，方便快速收集报告。
· 支持上传文档功能，将自己收藏的文档发送客服换取报告下载。
