# 腾讯元宝

概述

腾讯元宝电脑版是一款可深度思考的全能AI助手，提供「DeepSeek+」智能新体验，元宝期望通过AI能力帮助用户在逻辑推理、职场办公、知识学习、趣味创作、生活百科等多个领域提高效率和生活辅助。

功能简介

● AI阅读，文档精读助手
支持识别PDF、Word、PPT、Excel、TXT等36种文件格式；支持本地或微信文件上传，总结文档要点,获得图文并茂的精读体验。

● AI写作，灵感速达
快速写报告、写方案、写文案、写代码等，专业输出，给你更多灵感，内容创作更轻松。	

● 代码能力升级，新增多种开发语言运行支持
支持双栏展示，新增支持Python、C++、C、Java、JavaScript、Go、Fortran等开发语言在线运行，无需专门部署、即时验证代码效果

● 直连腾讯文档
支持腾讯文档导入精读与导出存储，实现内容无缝流转。

● 划词捕捉内容
划选文本一键翻译与搜索，全局划词快速响应，连续划词效率高，迷你对话窗直显，操作流畅无需切换界面。

● 支持朗读，音效任选
多重音效带来更生动的使用体验，同时支持播放倍速调控，朗读速度由你掌控。

● 截屏识图 智识万物
支持截图和多张图片上传，深度思考与图片识别强强联手，上传即可识别万物，如艺术鉴赏及多语种翻译。

● AI搜索，智搜一触即达
支持文字、图片、文档输入进行搜索，支持微信公众号、视频号等更多信源搜索。搜索结果涵盖图片、视频、音乐、地图等多模态形式，提供高效、多元的搜索体验。

● AI画图 秒出大片
支持AI画图，智能扩写更完整、清晰的提示词，快速生成高质量图像，释放无限想象。

● 历史问答，一键直达
支持快速定位历史问答，提升查询效率。

● 分组定制指令，秒变行业专家
通过个性化指令库，构建垂直领域AI专家矩阵，轻松解决复杂难题。
