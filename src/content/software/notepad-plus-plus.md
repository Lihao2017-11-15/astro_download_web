## 软件介绍

![Notepad++ 界面](https://cdn.jsdelivr.net/gh/Lihao2017-11-15/astro_download_web@main/public/screenshots/notepad-plus-plus-main.png)

Notepad++ 是一款免费的源代码编辑器和 Windows 记事本的替代品，支持多种编程语言。它基于强大的编辑组件 Scintilla，用 C++ 编写，使用纯 Win32 API 和 STL。

## 主要特性

- **语法高亮**：支持 80+ 种编程语言
- **代码折叠**：可折叠代码块，便于浏览
- **多文档界面**：同时编辑多个文件
- **正则表达式搜索**：强大的查找替换功能
- **插件支持**：丰富的插件扩展功能
- **轻量级**：启动快速，占用资源少

## 系统要求

- Windows 7/8/10/11
- 32位和64位版本
