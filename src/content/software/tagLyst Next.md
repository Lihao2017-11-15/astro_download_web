# tagLyst Next

概述

tagLyst Next 是一个标签式的文件资料管理利器，可自定义标签规则，智能归类文件，高效整合资料。性能优异很便利，专业功能可扩展，让每份资料都能有序管理。

功能简介

· 标签式文件管理给文件贴标签，支持按目录 / 类型 / 时间 / 同义词匹配等标签智能检索、分类；
· 分门别类的资料库，不同文件将归类到不同资料库，可添加多样化标签色彩 / 图标，丰富标签属性；
· 多种视图清晰布局，提供卡片流 / 阅读流 / 瀑布流 / 统计视图等多种视图展示，文件位置快速获取，便捷文档格式预览；
· 支持多种主流文件格式，无需打开即可一键预览图片 / 文字 / 办公文档等文件；
· 满足离线和云端需求，可完全离线使用，也可搭配第三方云服务、局域网内与团队共享标签和知识。
