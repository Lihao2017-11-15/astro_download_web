# Koala

* 开发者提供使用文档：https://github.com/oklai/koala/wiki#faq

概述

开源免费的预处理器语言图形编译工具。支持less、sass、coffeescript、compass实时编译。

功能简介

· 多语言支持：支持Less、Sass、CoffeeScript 和 Compass Framework
· 后台自动执行编译，还可设置各个语言的编译项
· 支持为项目创建一个全局配置，为文件设置统一编译选项
· 可跨平台运行，完美兼容Windows、Linux
