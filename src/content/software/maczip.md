# MacZip

概述

MacZip Mac版是Mac电脑上的一款专为Mac系统设计的压缩解压软件，新的MacZip现已完美兼容最新 Big Sur 系统。

功能简介

·支持超过 20 种压缩格式，完美处理加密解密，中文乱码等问题；
·轻松保护私密文件，加密后安全更省心。可随意丢到第三方云盘，再也不怕隐私泄露；
·可以直接对预览文件进行AirDrop，分享，运行服务或选择应用打开；
·快速过滤图片、文档。输入关键字，搜索包内文件。

## 推荐语

超级好用且安全的压缩工具。
