# zFuse

概述

一款非常不错的国产播放器，支持MP4、MOV、MKV等不同视频播放格式，支持同时播放多个媒体文件，同时也支持内嵌/外挂字幕，硬件加速，倍速播放等常见功能。

功能简介

· 支持MP4、MOV、MKV等不同视频播放格式
· 支持内嵌/外挂字幕
· 支持同时播放多个媒体文件
· 支持按照不同倍速进行播放视频
