## 软件介绍

7-Zip 是一款免费开源的文件压缩软件，拥有极高的压缩比。它支持 7z、ZIP、RAR、GZIP、BZIP2、TAR 等多种格式。

## 主要特性

- **高压缩比**：7z 格式提供比 ZIP 高 30-70% 的压缩率
- **支持多种格式**：7z、ZIP、GZIP、BZIP2、TAR、RAR、CAB、ISO 等
- **AES-256 加密**：支持强加密保护文件
- **集成 Windows 资源管理器**：右键菜单快速压缩解压
- **命令行版本**：支持脚本和自动化

## 系统要求

- Windows XP/Vista/7/8/10/11
- 64位和32位版本可选

2. 运行安装程序
