# xiaopiu

概述

xiaopiu是产品原型设计工具和团队实时协作平台，作为产品经理和交互设计师工作学习的必备软件工具，堪称快速原型交互设计神器。可制作手机app、web网站网页、桌面应用软件的原型，拥有海量原型交互模板资源素材，可替代axure在线使用无需下载

功能简介

· 海量云端资源，数百套精品原型、数千个典型页面、数百个常用组件，一键复用提升原型设计效率
· 多人协同，毫秒级传输，即使是在同一页面编辑也毫无压力
· 支持网页版，PC端，微信小程序，APP多平台实时预览
· 支持在线生成HTML链接与二维码，分享预览更方便。
