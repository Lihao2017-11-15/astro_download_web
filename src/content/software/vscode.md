## 软件介绍

![VSCode 界面](https://cdn.jsdelivr.net/gh/你的用户名/software-images@main/software-screenshots/vscode-main.png)

Visual Studio Code 是一款由微软开发的免费、开源的代码编辑器。它支持 Windows、macOS 和 Linux 平台，内置了对 JavaScript、TypeScript 和 Node.js 的支持，并拥有丰富的插件生态系统。

## 主要特性

- **智能代码补全**：IntelliSense 提供智能的代码补全、参数提示等功能
- **内置调试器**：支持断点调试、变量查看、调用堆栈等功能
- **Git 集成**：内置 Git 版本控制功能
- **扩展市场**：丰富的插件生态，支持各种编程语言和框架
- **终端集成**：内置终端，无需切换窗口

## 系统要求

- Windows 10/11
- macOS 10.15+
- Linux (Ubuntu 18.04+, Debian 9+, Fedora 30+)

## 安装说明

1. 下载对应平台的安装包
2. 运行安装程序
3. 按照向导完成安装
