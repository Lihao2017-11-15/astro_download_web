# Zoc Terminal

概述

Zoc Terminal是一款免费、跨平台的专业终端连接工具和Telnet/SSH客户端，能够通过SSH、Telnet、串行线缆以及ISDN等多种通信方式远程访问主机，并且支持行标签定制、历史命令记录、回溯、多窗口使用、命令自动补全、文件上传和下载等多种功能，非常适合开发或运维等专业人士使用。

功能简介

· 能够模拟ANSI、VT52、VT100、VT102、VT220、IBM 3270 和Secure Shell 等多种终端
· 可以通过SSH、Telnet、串行线缆以及ISDN等多种通信方式远程访问主机
· 支持行标签定制、历史命令记录、回溯、多窗口使用、命令自动补全、文件上传和下载等多种功能
· 支持SCP、Zmodem、Xmodem、Ymodem、Kermit等多种主要文件传输协议
· 支持高度个性化配置，可对工具栏、按钮、字体、颜色等按个人喜好进行设置

## 推荐语

ZOC Terminal 是一款Mac上强大的SSH/Telnet终端工具，相比系统自带的终端程序，ZOC Termina具有各种强大的功能，类似于SecureCRT，具有行标签定制，历史命令，回溯，多窗口，自动补全命令，文件上传和下载等特性，个人感觉比SecureCRT好用。
