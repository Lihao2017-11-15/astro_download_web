# OpenInTerminal

概述

是一款快速在终端中打开指定目录的效率工具，可以实现Finder中点击该应用图标就能在终端中打开该目录，同时支持使用在状态栏或快捷键打开默认终端和编辑器，可以说是开发者的利器。

功能简介

· 终端支持苹果默认终端, iTerm, Hyper 和 Alacritty
· 编辑器支持 文本编辑, Visual Studio Code, VSCode Insiders, Atom, Sublime Text等主流编辑器
· 支持键盘快捷键打开默认终端和编辑器
· 支持状态栏或Finder工具栏打开终端


## 推荐语

看到有人推荐 Go2Shell , OpenInTerminal 是国人开发，并且在 Github (https://github.com/Ji4n1ng/OpenInTerminal) 上开源的一款更加强大的应用，在 Finder 中你可以选择在终端还是编辑器中打开当前目录或者文件。
