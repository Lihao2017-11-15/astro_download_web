# Trello

概述

Trello是一款轻量级团队流程协作和列表管理平台，Trello灵活性高，可视化，可以和任何人一起管理任何事情。

功能简介

· 为即时反馈发表评论；
· 从计算机、Google Drive、Dropbox、 Box以及OneDrive上传文件；添加清单、标签、到期日等；
· 通知可确保你随时了解发生的大事；
· 在新建项目时，我们有多种项目模板可供选择；
· 多端可以同步，随时随地跟踪同步进展。

## 推荐语

管理项目不再需要冗长的邮件线程、老掉牙的工作表、 不怎么方便的便利贴以及繁杂的软件。Trello 让你一眼即可掌控项目中的一切。
