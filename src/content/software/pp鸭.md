# PP鸭

概述

PP鸭整合了业内最优秀的数种开源的图片压缩算法，会自动根据图片特征自动选择压缩参数。只需要将图片拖入PP鸭，就能自动批量压缩，省时省心。

功能简介

·一键操作，你只需要将图片拖拽至窗口上部，无需额外操作。
·更好的压缩效果，经过长时间反复调校和对比各种算法，帮你定位完美的平衡点。
·支持多种常用图片格式，能很好满足大家日常的图片压缩需要。
