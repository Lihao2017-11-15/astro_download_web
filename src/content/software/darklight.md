# DarkLight

概述

DarkLight是一款协助你轻松切换暗黑主题的软件，只需轻点状态栏就可完成切换～

功能简介

· 支持将图标显示在Touch Bar上，轻轻一点即可切换


## 推荐语

快速在 macOS 的 深色模式 和 浅色模式 之间切换！由一名医学生开发，支持状态栏图标、全局键盘快捷键、在线更新、Touch Bar 等！
