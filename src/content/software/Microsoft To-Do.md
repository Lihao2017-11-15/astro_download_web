# Microsoft To-Do

## 软件简介

“微软推出的一款跨平台任务管理应用。界面简单、支持iOS、安卓等多平台，还可导入奇妙清单数据、整合Outlook任务、更换主题，爱不释手。”

## 详细介绍

* 首次使用需要登录/注册账号

概述

Microsoft To-Do（微软待办）是微软推出的一款跨平台任务管理应用。无论是想提高工作效率、缓解压力，还是只是想少记点东西，微软待办均让你能够轻松规划你的一天、管理你的生活。

功能简介

· 支持与朋友、家人、同事共享列表和任务
· 多端同步，任意设备随身携带列表
· 支持同步Outlook
· 可设置截止时间和重复提醒
· 将任务分解成可管理的步骤
· 支持向任务添加备注、文件等
· 支持小视图窗口，随时跟踪待办任务

## 推荐语

Microsoft To-Do（微软待办）是微软在收购奇妙清单后推出的一款跨平台任务管理应用，支持 iOS、Android、Windows 10、Web平台，界面简介、功能简单易用，可以导入原奇妙清单数据、整合了Outlook任务、更换主题。支持 Siri、OK Google、3D Touch。在macOS平台有很多清单待办类应用中，但大多数高级功能都需要收费，而Microsoft To-Do是免费的，跨平台、简洁易用、整合outlook这些都是让我爱不释手的原因。
