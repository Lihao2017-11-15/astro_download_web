# Sequel Pro

* 入门使用说明： https://sequelpro.com/docs/ref/core-features；

概述

轻量简洁的数据库管理软件。可以一次性连接多个数据库，快速访问经常需要访问的数据库。

功能简介

· 完全支持MySQL数据库
· 支持访问本地数据库或者远程服务器上的MySQL数据库
· 支持通过 Command + R 快捷键，在[查询]中运行命令
· 支持在表格中枚举字段
