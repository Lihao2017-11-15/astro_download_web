# Mark Text

概述

Mark Text 是一款所见即所得的Markdown编辑器

功能简介

· 所输即所见，摒弃众多 Markdown 编辑器左边写作右边预览的写作方式，将编辑和预览窗口融为一体
· 支持 CommonMark Spec 和 GitHub Flavored Markdown Spec 语法格式，支持复制 Markdown 格式文本到其他 App 或浏览器
· 支持快捷键设置段落及行内样式，提高编辑效率
· 支持导出 HTML 和 PDF
