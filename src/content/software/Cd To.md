# Cd To

概述

Cd To Mac版是一款很方便的Mac终端工具，Mac用户可以在电脑上当前位置一键打开终端，节省了输入一大段地址的时间，大大提高了查找终端的速度。

功能简介

· 在任意Finder打开目录下，直接点击Cd To图标即可打开terminal并指定到该目录
· 功能简单实用，占用体积小

安装卸载说明

· 下载后直接解压
· 将解压后得到的"Cd To"程序拖拽至"应用程序"文件夹
· 在"应用程序"文件夹中找到"Cd To"应用，按住Command + Option 键，将应用图标拖拽到Finder（访达）的工具栏上

## 推荐语

Cd To Mac版是一款 macOS平台的可以帮助我们在当前位置打开终端的工具，避免了你在终端中需要输入一大串地址的困扰，Cd To Mac版让你直接右键就可以在菜单栏中找到终端，非常方便快捷。
