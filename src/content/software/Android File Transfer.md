# Android File Transfer

概述

是一款Mac上管理Androi手机的文件的软件，只需在连接手机前运行 Android File Transfer ，再连接手机，那么 Mac 就会识别出 Android 设备，并可以对里面的文件进行复制、粘贴、删除等管理。

功能简介

· 支持Mac快速连接Android 手机
· 支持从Mac里对Android手机文件复制、粘贴和删除

## 推荐语

Android File Transfer是一款帮助用户在Mac上读取安卓手机上的文件的Mac软件，Android File Transfer由于 Mac OS X 本身并不支持 MTP 协议，所以通过 USB 将 Android 设备连接到 Mac 电脑上是无法识别的，更别说读取里面的文件了，但是它能解决这个问题。软件非常简洁！Google 官方出品，不含有任何广告、收费等，几乎适配所有 Android 手机，适用于 macOS 10.7 版本或更高
