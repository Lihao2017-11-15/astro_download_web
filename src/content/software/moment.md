# Moment

概述

Moment 是一款常驻于 macOS 菜单栏和通知中心里的倒数日应用，美观小巧不打扰，每一天都值得期待！

功能简介

·任务有进度条，你的生活也应该有一个时间进度条。通过 Moment 了解此时刻在今日的进度、知道今年已经完成了多少进度。
·支持对指定日期添加对应的 Emoji 和照片。根据事件类型不同，支持每年、每月、每周、每天循环创建。重要日子临近时，还可以自定义提醒推送的时间，选择提前通知.
·每个日期都可被单独添加到菜单栏，不用打开 Moment 随时可在菜单栏查看。也可将事件单独添加到通知中心「今天」小组件，查看天气的时候，也可以看看那些大日子现在是什么进度了。

## 推荐语

好用的倒数日时间进度查看和纪念日提醒工具。
