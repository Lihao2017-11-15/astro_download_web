# 字由

概述

字由是为设计师量身定做的一款字体工具，拥有丰富的字体库和大量的免费商用字体，满足各种设计场景需求，并且支持与多种设计软件协作，轻松应用字体，实用又便捷！

功能简介

· 拥有丰富的字体库和大量的免费商用字体，满足各种设计场景需求；
· 可以通过收藏、搜索、标签、案例等快速找到心仪的字体，批量预览对比不同字体样式；
· 字体一键激活无需下载，可与设计软件的协作，轻松预览应用字体；
· 可收藏字体及自定义分组，自动读取设备中的本地字体，上传字体文件至客户端。
