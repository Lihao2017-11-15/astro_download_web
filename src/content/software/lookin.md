# Lookin

* 第一次使用时，需要把Lookin的iOS框架嵌入在你的iOS项目中：https://lookin.work/faq/integration-guide/

概述

Mac平台上一款iOS UI 调试软件，可以在Mac上查看与修改 iOS App 里的 UI 对象。

功能简介

· 显示变量名，以及 indexPath 等各种提示；
· 按住“Option”键，随时测距；
· 可随时访问iOS App 里的任意 NSObject 的属性或方法；
· 即使没有Mac，也可以单独运行在你的iPhone或iPad上；

## 推荐语

开发 iOS App 时的 UI 调试软件，相比于 Xcode 内置工具，多了“变量名显示”、“测距”、“文件导出”等一堆实用功能，甚至可脱离 Mac 单独运行在 iPhone 或 iPad 上，免费又好用。
