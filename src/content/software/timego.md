# TimeGo

概述

TimeGo是一款Mac平台上的轻量计时提醒工具，能够设置计时时长和具体通知信息并可以组合为番茄钟，支持语音提醒功能，拥有多国语言供用户选择，工具的颜色主题跟随系统。

功能简介

· 界面美观简洁，颜色主题跟随系统变化
· 可设置倒计时的时长和具体的通知信息，并可以多个倒计时组合一起作为番茄钟
· 支持语音提醒功能
· 支持多国语言
