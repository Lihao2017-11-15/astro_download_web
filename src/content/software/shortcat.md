## 详细介绍

概述

Shortcat是一款键盘辅助工具，可以通过快捷键到达屏幕上的任何文字而不需要移动鼠标，使用起来可以帮助用户增快做事的效率，用户只需要使用多种组合键就能轻松的进行办公。

功能简介

· 无需使用鼠标即可点击！Shortcat 特别适合快速打字员。只需使用键盘快捷键激活 Shortcat，键入几个与你要单击的 UI 元素匹配的字母，Shortcat 将在当前活动窗口（和菜单栏）中搜索匹配元素；
· 利用了 Mac OS X 附带的所有应用程序和大多数应用程序都支持的辅助功能 API；
· 将手从键盘移到鼠标，单击一次，然后再移回键盘，平均需要 3 秒钟的时间。Shortcat 可让你将手放在键盘上单击，从而节省时间和精力。！

## 推荐语

按下快捷键 Cmd+Shift+Space，输入屏幕上的任何文字，光标就会移动到此处，多选使用 control + 字母选择，回车确认，非常方便。
