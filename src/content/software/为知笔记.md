# 为知笔记

概述

为知笔记是一款国产的笔记记录、资料收集整理工具。它支持一键收藏、全文检索、多级目录以及 Markdown 语法等功能.

功能简介

·为知笔记强大的群组管理、多级文件夹、@提及和评论、多人编辑、离线使用、全文检索、多端同步等创新功能，可以满足多种场景需求；
·和团队成员一起使用，分享工作资料、收集成员意见、搭建团队知识库、发送通知公告；和客户一起使用，为客户提供全平台资料服务；和家人一起使用，共享家庭资料。

## 推荐语

为知笔记是一款多平台界面简洁、操作简单的云笔记工具，几乎没有使用方面的困难。
