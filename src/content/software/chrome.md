## 软件介绍

![Chrome 浏览器](https://cdn.jsdelivr.net/gh/Lihao2017-11-15/astro_download_web@main/public/screenshots/chrome-main.png)

Google Chrome 是由 Google 开发的免费网页浏览器，以其速度、安全性和简洁的界面而闻名。它是目前全球使用最广泛的浏览器之一。

## 主要特性

- **极速浏览**：V8 引擎提供快速的 JavaScript 执行速度
- **安全保护**：内置恶意软件和钓鱼网站防护
- **同步功能**：跨设备同步书签、密码和历史记录
- **扩展程序**：Chrome Web Store 提供丰富的扩展
- **隐私模式**：无痕浏览保护隐私

## 系统要求

- Windows 10/11
- macOS 10.15+
- Linux
- Android / iOS
