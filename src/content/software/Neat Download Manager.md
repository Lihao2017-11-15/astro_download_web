# Neat Download Manager

概述

Neat Download Manager Mac版是Mac上免费的下载管理器。可以加快下载速度，直到达到你的总可用带宽。

功能简介

· 支持根据下载状态和文件类型归类下载的文件夹
· 下载中同样支持设置当前下载的带宽限制
· 支持在下载时设置最大连接数
· 支持32线程
· 拖拽需要下载的文件到状态栏图标，即刻开始下载
· 具有浏览器插件，可以捕获、检测网站上的视频、音频链接
· 可以下载HLS视频的所有.ts文件，最后将所有片段合并为一个.ts文件
