# Firefox

概述

Firefox是一款引擎反应更快,内存占用更少的浏览器，支持数以千计的浏览器扩展，可为浏览器设置不同的主题，Firefox 可以自动帮您导入 Chrome 上的书签、自动填写的表单、密码以及偏好设置等。

功能简介

· 账户同步：支持同步书签、自动填写的表单、密码以及偏好设置等
· 浏览器拓展：支持 LastPass、Evernote 等数以千计的扩展
· 官方内置不同的浏览器主题，可自由定制
· 隐私保护：Firefox 会在您上网时拦截追踪器，也能在浏览结束后自动清除浏览记录
