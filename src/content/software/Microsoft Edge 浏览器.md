# Microsoft Edge 浏览器

概述

Microsoft Edge 是微软基于 Chromium 开源项目全新打造的新一代浏览器，具有性能强，速度快，界面清爽的特点

功能简介

·  支持多标签浏览，每个标签互相独立不影响 
·  内置扩展商店，同时也支持安装Chrome拓展插件
·  登陆微软账户，就可以在各个设备上的 Edge 中，同步收藏夹、扩展、历史记录、设置以及打开的标签页等
