# 极度扫描

## 软件简介

“极度扫描是一款跨平台的 OCR 工具，支持表格 / 公式识别、图片转文字，并且可以通过截图 / 拍照 / 选择图片 / PDF 等多种方式导入，轻松助你提升办公效率。”

## 详细介绍

概述

极度扫描支持扫描，文字识别，支持每天20次的mathpix公式识别，超高的准确度和强大的编辑模式。可以对图片上的阴影进行最大程度的抑制，保证达到最好的效果。

功能简介

· 多种导入方式：根据需要选择不同的方式导入图片或 PDF 进行识别，还能够对导入的内容进行裁剪，只留下需要识别的选区。并且，软件支持为图片添加增强、灰度和黑白效果，从而提高识别内容的准确性；
· 简易模式：极度扫描不仅支持图片转文字，而且支持表格识别和公式识别。更重要的是，
· 文档模式：文档模式下，可以对扫描结果进行编辑，并且能够对其进行更换字体、大小标题切换、插入表格、调整文档方向、设置对其方式等自定义操作，还支持插入表格和超链接。
· 专业模式：可进行更强大的文档编辑操作，除拥有文档模式的大部分功能之外，还支持实时翻译多种语言。
