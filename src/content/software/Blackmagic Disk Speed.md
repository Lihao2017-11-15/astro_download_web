# Blackmagic Disk Speed

“是一款Mac平台的硬盘读写速度测试工具 ，可以快速测试本机硬盘或外接硬盘、U盘等的读写速度，评估硬盘性能和可读性    ”

## 详细介绍

概述

是一款Mac平台的硬盘读写速度测试工具 ，可以快速测试本机硬盘或外接硬盘、U盘等的读写速度，评估硬盘性能和可读性

功能简介

· 支持检测并验证硬盘性能，以及硬盘读入和读出的能力，只需按下开始键就可以测试，简单实用
· 主窗口的底部，您可以查看磁盘可以处理的视频格式和测试结果（对于每种文件格式，以每秒帧数表示）
