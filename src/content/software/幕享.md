# 幕享

概述

幕享Mac版是为Mac量身定制的免费无线投屏软件。支持各种智能设备与Mac同屏，包括iOS、Android设备和Mac电脑的相互投屏；Mac画面投射到Windows电脑、智能电视/电视盒子、投影仪等智能设备。

功能简介

· 支持iOS、Android设备和Mac电脑的相互投屏；
· 支持Mac画面投射到Windows电脑、智能电视/电视盒子、投影仪等智能设备；
· 使用苹果官方AirPlay技术，屏幕镜像流畅无延迟，音画同步；
· 投屏至Mac电脑后，可以对投屏画面进行录制，对投屏资料内容进行圈点、书写、涂鸦等。
