# MagicaVoxel

* 解压后直接打开应用时黑屏，需将应用移动到其他任意位置，再拖回原位置，重新打开即可正常显示。

概述

Magicavoxel是一款开源免费的轻量级8-bit体素编辑器，具有媲美C4D的渲染效果；操作简便，轻松实现像素风，乐高风等一系列卡通风格设计效果～

功能简介

· 可创建和编辑3D，8位模型
· 自带渲染工具：可以呈现不同的光线条件、光照度、明暗度和锐度
· 支持多种画笔工具，随时画出你理想中的效果
· 内置一些基础模型，还支持插入预设形状及图案
