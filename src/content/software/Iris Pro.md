# Iris Pro

概述

在长期使用电脑的过程中，免不了要受到屏幕发出的蓝光等对眼睛的伤害，如何尽可能地降低蓝光带来的危害，保护用眼视力就成了我们需要考虑的问题。Iris Pro 能够调节屏幕颜色和亮度，降低蓝光，从而减少对眼睛造成的刺激和伤害，改善用户睡眠。

功能简介

·Iris Pro 拥有多种启动模式和多种场景组合搭配，满足不同时间段使用电脑的各种需求，也避免了在工作时段对屏幕显示带来干扰；
·通过设置不同的计时器，iRis Pro 会在计时结束后进行相应的休息或重新投入工作，非常贴心；
·除预设的场景，还可以通过软件的高级设置选项，自定义各个选项的详细参数，以满足更加复杂的需求。

## 推荐语

人眼会通过夜间蓝光量较少来分泌褪黑素刺激睡意， Iris Pro 还起到帮助改善睡眠质量的作用。
