# MuMu模拟器

概述

Mac上一款安卓模拟器、精品手游服务平台。安装后可在电脑上运行各类游戏和应用，操作流畅、功能全面、智能兼容，可兼容市面上90%的主流手游，ROM版本为6.3.10.1。

功能简介

· 支持《明日方舟》、《和平精英》等市面上常见手游，Mac专属特供版
· 可在应用中心随时下载应用或游戏，支持市面上90%的应用
· 支持多开应用
· 支持截屏、操作录制、文件共享、摇一摇等
· 支持手动安装应用，下载对应应用的apk文件，点击加入即可安装
