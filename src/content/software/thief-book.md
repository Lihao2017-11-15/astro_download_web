# Thief-Book

概述

一款可以在任务栏、桌面、TouchBar上进行摸鱼的最强的跨平台摸鱼神器，支持小说、股票、按键翻页、鼠标翻页、自动翻页让你体验不同的工作 996，上班族必备神器

功能简介

· 支持小说和股票两种摸鱼玩法，小说导入需从Mac启动台找到应用，点击进入后选择设置，在小说路径导入
· 自定义透明背景，随意调整大小，完美融入各种软件界面
· 三个快捷键，实现完美的摸鱼
· 跨平台支持 Mac和Windows
