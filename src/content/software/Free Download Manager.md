# Free Download Manager

概述

Free Download  Manager 是Mac上一款无广告，全免费的mac下载器。

功能简介

· 支持BT
· 支持边下边播（实时预览）
· 可抓取网页链接、图片、文件等，下载整个网页内容
· 多线程、加速下载，提高下载效率
· 断点续传、接续下载
· 可归类整理下载任务，支持预下载
· 支持限速，下载同时不影响看视频
