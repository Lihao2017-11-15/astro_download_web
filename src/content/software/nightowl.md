# NightOwl

* 开发者会利用Google Analytics匿名收集数据用以改善应用体验，若不需求可在应用设置中取消勾选“Send usage Statistics”。

概述

帮助你轻松切换深色主题（暗黑模式）的极简状态栏应用。

功能简介

· 状态栏轻点，一键切换
· 可自定义时间设置切换
· 支持根据当地日落时间切换
· 可对应用设置主题切换

## 推荐语

NightOwl 是一款可以让你快速切换系统浅色、深色外观主题的小工具，用起来非常的简单方便。
